# Warbly's

Flappy bird clone. Created using a tutorial on YouTube. Added my own functionality and, of course, an inspired paint job. 

It took me two years to upload this project because I couldn't figure out how to scale windows across devices. AI was able to explain the solution in 20 minutes. 

Windows will advise you to not run it but I pinky promise it's okay. I'm not smart enough to create a virus. ¯\_(ツ)_/¯
